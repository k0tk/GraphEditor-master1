# GraphEditor
![ScreenShot](http://dl1.joxi.net/drive/0002/2979/134051/150606/aa90be62e4.jpg) 
##Дипломная работа по разработке визуального редактора для построения гиперграфов.
### Основные возможности:
* Сериализация/Десериализация гиперграфа с помощью JAXB;
* Построение гиперграфов
* Работа со множеством гиперграфов в различных вкладках
* Выполнение алгоритмов на построенном гиперграфе
* Zoom рабочей области
* Построение списка доступных файлов для открытия
